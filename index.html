Welcome to Boot Navigator homepage

Boot Navigator is FREEWARE program which allows you boot operating system (OS) from any of your primary partitions. Currently supported partition types are:
  • FAT12 - DOS
  • NTFS - Windows NT
  • FAT32 - Windows 9x
  • 4F - QNX
  • Linux Ext2 - Linux
If you want to boot from any other partition just change code of Boot Navigator because source code is distributed with program.
This program works in such a way:
  • Finds all known (mentioned above) partitions and displays them.
  • User can choose (by pressing 1, 2 or other numeric key) from which partition to boot.
  • If user do not choose within 6 seconds Boot Navigator boots first OS.
